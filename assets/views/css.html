CSS Introducción.

Junto a HTML, CSS es el lenguaje más importante para el diseño web. Quien quiera crear páginas web modernas no podrá pasar por alto las hojas de estilo en cascada (Cascading Style Sheets). De la misma forma, contar con conocimientos básicos en CSS resulta de gran ayuda a la hora de trabajar con editores gráficos HTML, sistemas modulares de diseño web o sistemas de gestión de contenidos.

Un poco de Historia

  • Alrededor de 1970 aparecieron las hojas de estilos aparecieron poco después.
  • A finales de 1994 y 1995 Lie y Bos se unieron para definir un nuevo lenguaje y lo llamaron CSS (Cascading Style Sheets).
  • En 1995, el W3C decidió apostar por el desarrollo y estandarización de CSS y lo añadió a su grupo de trabajo de HTML.
  • A principios de 1997, el W3C decide separar los trabajos del grupo de HTML en tres secciones: el grupo de trabajo de HTML, el grupo de trabajo de DOM y el grupo de trabajo de CSS.
  • En 1998, el grupo de trabajo de CSS publica su segunda recomendación oficial "CSS nivel 2".
  • El primer navegador con soporte completo de CSS 1 fue la versión para Mac de Internet Explorer 5, que se publicó en el año 2000.
  • Por el momento, ningún navegador tiene soporte completo de CSS 2.1.

    Soporte de CSS en los navegadores.

    Internamente, los navegadores están divididos en varios componentes. La parte del navegador que se encarga de interpretar el código HTML y CSS para mostrar las páginas se denomina motor. Desde el punto de vista del diseñador CSS, la versión de un motor es mucho más importante que la versión del propio navegador.

    Selectores CSS.

    CSS soporta diferentes selectores que permiten una asignación personalizada de instrucciones de diseño. Sin embargo, para comenzar es suficiente con familiarizarse con los selectores de ID, de clases, de tipo o etiqueta y universales.

    Integrar CSS en HTML.

    Para que un navegador pueda considerar el formato CSS como parte de la estructura de la página web, las instrucciones de diseño deben estar vinculadas al código fuente HTML.

  • Su integración en las etiquetas HTML:

    <h2 style=": red;">Subtítulo</h2>

  • Su inclusión en el encabezado HTML

  • El enlazamiento a una hoja de estilo externa.

    • <link rel="stylesheet" href="ejemplo.css">

    Esta última opción es considerada la más idónea: en la práctica las hojas de estilo se crean generalmente como archivos de texto externos y son integradas posteriormente.
