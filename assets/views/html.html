HTML Introducción.

El HTML define una estructura básica y un código (denominado código HTML) para la definición de contenido de una página web, como texto, imágenes, videos, juegos, entre otros. Es un estándar a cargo del World Wide Web Consortium (W3C) o Consorcio WWW, organización dedicada a la estandarización de casi todas las tecnologías ligadas a la web, sobre todo en lo referente a su escritura e interpretación. HTML se considera el lenguaje web más importante siendo su invención crucial en la aparición, desarrollo y expansión de la World Wide Web (WWW). Es el estándar que se ha impuesto en la visualización de páginas web y es el que todos los navegadores actuales han adoptado.

Un poco de Historia

  • En 1980 el físico Tim Berners-Lee propuso un nuevo sistema de “hipertexto” para compartir documentos.
  • En 1991 se publicó el primer documento formal con la descripción de HTML bajo el nombre “HTML Tags”.
  • En 1993 se presenta la primera propuesta oficial para convertir HTML en un estándar se realizó en 1993.
  • En 1995, el organismo IETF publicó el estándar HTML 2.0.
  • A partir de 1996, los estándares de HTML los publica el W3C.
  • En 1998 se publicó el estándar HTML 4.0.
  • En el año 2004, las empresas Apple, Mozilla y Opera se aliaron.
  • En el año 2008 se publica el primer borrador para el estándar HTML5.
  • HTML5 ya es un estándar recomendado por el consorcio W3C.

    • HTML Editores.

    Las páginas web se pueden crear y modificar utilizando editores HTML profesionales. (Visual Studio Code, Sublime Text...) Sin embargo, para aprender HTML, se recomienda iniciar utilizando un editor de texto simple como el Bloc de Notas, Notepad++ (PC). Es una buena forma de iniciar, ya que estos editores no tienen ninguna ayuda, por lo que el estudiante comprenderá mejor el concepto de Lenguaje de Marcado.

    HTML Ejemplos Básicos - Documentos.

    Todos los documentos HTML deben comenzar con una declaración de tipo de documento: <! DOCTYPE html>. El documento HTML en sí mismo comienza con <html> y termina con </html>. La parte visible del documento HTML se encuentra entre <body> y </body>.

    HTML Elementos

    El elemento HTML es todo, desde la etiqueta inicial hasta la etiqueta final: <tagname> El contenido va aquí ... </tagname> Ejemplos de algunos elementos HTML:

    <h1> Mi primer título </h1>

    <p> Mi primer párrafo. </p>


    HTML Encabezados

    Los encabezados HTML son títulos o subtítulos que desea mostrar en el contenido de una página.

    <h1>Encabezado 1</h1>

    <h2>Encabezado 2</h2>

    <h3>Encabezado 3</h3>

    <h4>Encabezado 4</h4>

    <h5>Encabezado 5</h5>
    <h6>Encabezado 6</h6>
